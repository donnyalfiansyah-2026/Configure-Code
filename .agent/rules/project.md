Tujuan:

- Bantu membangun aplikasi web (JS/HTML/CSS) dengan Docker dan GitHub.
- Prioritaskan solusi sederhana, bisa dijalankan, dan mudah dipahami.

Perilaku kerja:

- Selalu mulai dengan ringkasan perubahan yang akan dibuat.
- Pecah pekerjaan besar menjadi langkah kecil yang bisa diverifikasi.
- Jangan menghapus file/folder kecuali diminta eksplisit.
- Jangan menambah dependency baru tanpa mengecek kebutuhan dan dampaknya.
- Setelah perubahan kode: jalankan lint/test/typecheck bila tersedia.
- Pastikan npm scripts konsisten dengan VS Code Tasks dan CI.

Konvensi output:

- Gunakan bahasa Indonesia.
- Beri referensi file path untuk setiap perubahan penting.
- Hindari menulis komentar di dalam kode kecuali diminta.

Keamanan:

- Jangan menaruh secret di repo (.env, token).
- Jangan log credential atau token.
